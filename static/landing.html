Digital Garage - Manage AI Workflows

Build AI agents, control their behaviour, connect them into collaborative workflows — all self-hosted.

Get Started View on GitHub

What you can do

Create Agents

Define agents with custom names, roles, system prompts, and models. Configure their personality and capabilities.

Configure Behaviour

Set schedules (Cron Jobs), memory scope, skills, interaction rules, and guardrails. Control how agents operate autonomously.

Orchestrate Workflows

Connect agents into collaborative workflows with conditions and feedback loops. Use pre-built templates or build your own.

Real Runtime

Agents run on actual infrastructure, execute real tools, and communicate asynchronously to complete tasks.

Email Integration

Connect agents to email accounts (IMAP/SMTP). They can receive, process, and respond to messages autonomously.

Web UI

Manage everything visually — agent creation, workflow design, monitoring, logs, and message history.

Architecture

Clear separation between UI, runtime, and persistence layers.

Get Started

Try these below options to run Digital Garage locally.

Option 1: Docker (recommended) 

$ git clone <repo-url>
$ cd digital_garage
$ cp .env.example .env
$ docker compose up -d --build

Option 2: Manual install 

$ git clone <repo-url> && cd digital_garage
$ python3 -m venv venv && source venv/bin/activate
$ pip install -r requirements.txt
$ python setup.py
$ uvicorn app:app --host 0.0.0.0 --port 7000

Test with Uvicorn 

$ uvicorn app:app --host 0.0.0.0 --port 7000 --reload

Opens at http://localhost:7000
