# Digital Garage - Acknowledgments

This project builds upon the work of many talented developers and open-source projects. We gratefully acknowledge their contributions.

## Core Dependencies

### AI/ML Frameworks
- **[Ollama](https://ollama.ai/)** - Local LLM inference server
- **[LangChain](https://github.com/langchain-ai/langchain)** - LLM orchestration framework (MIT License)
- **[ChromaDB](https://www.trychroma.com/)** - Vector database for embeddings
- **[Sentence Transformers](https://www.sbert.net/)** - Text embedding models

### Web Framework & Tools
- **[FastAPI](https://fastapi.tiangolo.com/)** - Modern Python web framework (MIT License)
- **[Uvicorn](https://www.uvicorn.org/)** - ASGI server (BSD License)
- **[SQLAlchemy](https://www.sqlalchemy.org/)** - Python SQL toolkit (MIT License)
- **[Pydantic](https://pydantic.dev/)** - Data validation (MIT License)

### Frontend Libraries
- **[CodeMirror](https://codemirror.net/)** - Code editor component (MIT License)
- **[Marked](https://marked.js.org/)** - Markdown parser (MIT License)
- **[DOMPurify](https://github.com/cure53/DOMPurify)** - XSS sanitizer (Apache-2.0/MIT)

### Email Integration
- **[imaplib](https://docs.python.org/3/library/imaplib.html)** - Python IMAP client (PSF License)
- **[smtplib](https://docs.python.org/3/library/smtplib.html)** - Python SMTP client (PSF License)
